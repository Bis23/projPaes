Guia de Instalação e Configuração

1. Instalação das Dependências
  MySQL2:
  "npm install mysql2"
  ou:
  "npm install mysql"

  Express:
  "npm install express"

  Body Parser (Talvez seja necessário):
  "npm install body-parser"

---

2. Configuração Inicial do Banco de Dados
  2.1 Criando o Banco de Dados: Execute o script localizado no arquivo "meu_banco.txt"

  2.2 Verifique no arquivo "conexaoDB.js" se os dados de conexao estao correnpondentes.

---
